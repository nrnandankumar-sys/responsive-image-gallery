Responsive Image Gallery

Overview

The Responsive Image Gallery is a frontend web project developed using HTML, CSS, and JavaScript. It provides an interactive and visually appealing gallery experience with lightbox functionality, image navigation, zoom effects, filters, and responsive design.

This project was created to improve frontend development skills and strengthen understanding of DOM manipulation, responsive layouts, and interactive UI design.

Functionalities

->Image Gallery

->Displays images in a responsive grid layout.

->Lightbox View

->Opens selected image in fullscreen mode.

->Navigation Buttons

->Users can move between images using Previous and Next buttons.

->Hover Effects

->Smooth zoom and brightness effects on image hover.

->Image Filters

->Filter images by categories such as:

- Nature
- Animals
- City

->Zoom Functionality

Users can zoom in and zoom out images inside the lightbox.
