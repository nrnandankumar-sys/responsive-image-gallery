# Responsive Image Gallery

## Overview

The Responsive Image Gallery is a frontend web project developed using HTML, CSS, and JavaScript. It provides an interactive and visually appealing gallery experience with lightbox functionality, image navigation, zoom effects, filters, and responsive design.

This project was created to improve frontend development skills and strengthen understanding of DOM manipulation, responsive layouts, and interactive UI design.

---

## Features

✅ Responsive Image Gallery Layout
✅ Lightbox Image Preview
✅ Next / Previous Navigation Buttons
✅ Hover Effects and Smooth Transitions
✅ Zoom In / Zoom Out Functionality
✅ Image Category Filters
✅ Mobile Responsive Design
✅ Clean User Interface
✅ Interactive JavaScript Functionality

## Functionalities

### 🖼️ Image Gallery

Displays images in a responsive grid layout with clean spacing and alignment.

### 🔍 Lightbox View

Opens selected images in fullscreen preview mode for better viewing experience.

### ⬅️➡️ Navigation Buttons

Users can move between images using Previous and Next buttons inside the lightbox.

### ✨ Hover Effects

Smooth zoom and brightness effects are applied when hovering over images.

### 🎯 Image Filters

Filter images by categories such as:

* 🌿 Nature
* 🏙️ City
* 🦁 Animals

### 🔎 Zoom Functionality

Users can zoom in and zoom out images inside the lightbox view.

### 📱 Responsive Design

The gallery automatically adjusts for mobile, tablet, and desktop screens.

### ⚡ Smooth Transitions

Interactive animations and transitions improve the overall user experience.
